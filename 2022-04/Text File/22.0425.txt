DB 관점에서 트랜잭션이란 무슨 뜻인가?
- 데이터베이스 관리 시스템에서 상호 작용의 단위이다.

DB 관점에서 RDBMS 와 DBMS 는 무엇이며 그 차이는 무엇인가?

전자정부프레임워크 Spring은 Java 기반이다.

Java 관점에서 Interface(인터페이스)란?
- 클래스들이 구현해야 하는 동작을 지정하는데 사용되는 추상형 규제이다.
- 클래스와 달리 다중 상속이 가능하다.
- 객체 내부를 알 필요 없고 인터페이스의 메소드만 알고 있으면 된다.

Java 관점에서 다형성이란?
- 하나의 메소드나 클래스가 상황에 따라 다양한 방법으로 동작하는 것이다.
- 오버로딩은 다형성의 가장 대표적인 예이다.

Java 관점에서 Generic(제네릭) 이란?
- 클래스 내부에서 사용할 데이터 타입을 나중에 인스턴스를 생성할때 확정하는것이다.

Back-End 관점에서 mutable, immutable 이란?
- mutable은 변경가능한 객체, 상태를 바꿀 수있는 객체이다.
- immutable이란 반대로 변경 불가한 객체, 상태를 바꿀 수 없는 객체이다.

Back-End 관점에서 OOP 란?
- Object-Oriented Programming 의 약어로써, 객체 지향 프로그래밍 이란 뜻이다.
- 캡슐화, 다형성, 상속 등등 특징을 가지고 있다.

Java 관점에서 Reflection 이란?
- 객체를 통해 클래스의 정보를 분석해 내는 프로그램 기법이다.

Spring 관점에서 Annotation 이란?
- 자바 소스코드에 추가하여 사용할 수 있는 메타데이터의 일종이다.
- @기호를 붙여서 사용한다.
- CompileTime, Runtime시에 해석될 수 있다.
- @Target으로 어노테이션이 적용할 위치를 선택한다.
- @Retention으로 컴파일러가 어노테이션을 다루는 방법을 설정한다.

Java 관점에서 Collection 이란?
- 데이터를 저장하는 기본적인 자료구조들을 한 곳에 모아 관리하고 편하게 사용하기 위해서 제공한다.
- 모든 Collection의 상위 인터페이스로써 Collection이 갖고 있는 핵심 메소드를 선언한다.
- List, Set, Map 등이 있다.

Java 관점에서 JVM 자바 가상 머신 이란?
- 자바 애플리케이션을 클래스 로더를 통해 읽어 들어 자바 API와 함께 실행하는 것이다.

Java 관점에서 Class Loader 
- 각 소스코드 파일을 오브젝트 파일로 변환시켜주는 역할을 한다.
- Linker = 모든 오브젝트 파일들을 하나의 오브젝트 파일로 합친다.

Java 관점에서 Runtime Data Areas
- JVM이 프로그램을 수행하기 위해 OS로부터 별도로 할당 받은 메모리 공간을 의미한다.
- 모든 쓰레드가 공유하는 메모리 영역이다.

Back_End 관점에서 쓰레드란 무엇인가?
- 프로그램(프로세스) 실행의 단위이다.
- 하나의 프로세스는 여러개의 쓰레드로 구성이 가능하다.
- 하나의 프로세스를 구성하는 쓰레드들은 프로세스에 할당된 메모리, 자원 등을 공유한다.
