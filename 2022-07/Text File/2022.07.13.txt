2. 스프레드시트 일반

2 - 1 엑셀의 화면 구성

- 빠른 실행 도구 모음, 리본 메뉴, 이름 상자, 수식 입력줄, 상태 표시줄

2 - 2 워크시트

- 데이터 작업이 이루어지는 기본 문서

- 워크시트 선택, 워크시트 이름변경, 워크시트 삽입, 워크시트 이동, 워크시트 복사, 워크시트 삭제

2 - 3 데이터 입력

- 자동 입력은 문자 데이터에만 적용되고 숫자, 날짜, 시간 데이터엔 적용 안된다.

- 시트와 관련된 작업(시트 이름 변경, 삽입 ,삭제 복사, 이동)은 실행 취소가 안된다.

2 - 4 데이터 형식

- 문자 데이터, 숫자 데이터, 날짜/ 시간 데이터, 수식 데이터, 한자, 특수 문자, 메모, 윗주

2 - 5 채우기 핸들을 이용한 연속 데이터 입력

- 숫자 데이터 : 1씩 증가, 첫, 두번째 값 증감 만큼 변함

- 사용자 지정 목록 : 목록에 등록된 순서대로 반복되어 입력

- 문자 데이터 : 동일한 데이터 복사

- 혼합 데이터 : 문자 그대로 숫자 숫자대로

- 날짜 데이터 숫자와 비슷

2 - 6 찾기

- 특정 내용 찾기

- 만능 문자 자체 검색하려면 물결 붙이기

2 - 7 셀 포인터 이동 / 범위 지정

2 - 8 파일 -> 옵션

- 일반 탭 : 사용자 인터페이스 옵션, 새 통합 문서 만들기

- 수식 탭 : 계산 옵션, 수식 작업

- 고급 탭 : 편집 옵션, 이 통합 문서의 계산 대상, 이 워크시트의 표시 옵션, 일반

2 - 9 선택하여 붙여넣기

- 잘라내기엔 안되고 복사한 경우에만 사용가능

2 - 10 데이터 유효성 검사

- 데이터를 정확하게 입력할 수 있도록 도와주는 기능

2 - 11 통합 문서

- 작성, 열기, 닫기

- 일반옵션 - 백어 파일 항상 만들기, 열기 암호, 쓰기 암호, 읽기 전용 권장

2 - 12 서식 파일

- 일정한 형식이나 스타일을 적용하여 만들어 놓은 문서이다.

-  일반 확장자는 xltx, 매크로 포함은 xltm

2 - 13 통합 문서 공유

- 하나의 통합 문서를 여러 사람이 공동 작업 할수 있게 한다.

- 조건부 서식, 차트, 시나리오, 부분합, 데이터 표, 피벗 테이블 보고서 등 변경 불가

- 하위 버전에서 사용 할 수 없다.

2 - 14 시트 보호/ 통합 문서 보호

- 시트 보호

. 값 변경 안되게함

- 통합 문서 보호

2 - 15 사용자 지정 서식

2 - 16 조건부 서식

- 규칙 작성 시 셀을 클릭하면 절대 참조로 지정됨

- 규칙별로 다른 서식 적용 가능

2 - 17 오류 메시지

- 순환 참조 경고 (재귀와 비슷)

2 - 18 셀 참조

- 상대 참조, 절대 참조, 혼합 참조

2 - 19 이름 정의

2 - 20 통계 함수
