API 란?

- 어플리케이션 프로그래밍 인터페이스란 뜻의 함축어이다.

RESTAPI 란?

- 분산 시스템을 위한 HTTP 기반 소프트웨어 아키텍처

- 웹 어플리케이션, 다양한 언어, 모바일 어플리케이션, 다른서버 등 서로 통신할 수 있도록
통역 역할을 해주는 API 이다.

CI/CD란?

Ci = 빌드, 테스트 자동화

CD = 배포 자동화

빌드란 무엇인가?

. 컴파일

. 실행 가능한 상태로 만드는 과정(산출물)

정적 ~ 은 거의 컴파일과 과련된 상태에서 관련있는것이다.

동적 ~ 은 거의 실행과 관련된 상태에서 관련있는 것이다.

자바는 JVM(자바가상머신)이 프로그램 실행 도중 필요한 클래스를 찾아서 클래스패스에
로드해주는데 이는 동적링크의 예이다.
