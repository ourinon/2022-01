Node.js 란?

Chrome V8 JavaScript 엔진으로 빌드 된 JavaScript 런타임이다.
즉, JavaScript 의 실행환경? 정도로 생각하자

1. Node.js는 JavaScript를 서버에서도 사용할 수 있도록 만든 프로그램이다.

2. Node.js는 V8이라는 JavaScript 엔진 위에서 동작하는 자바스크립트 런타임이다.

3. Node.js는 서버사이트 스크립트 언어가 아니고 프로그램(환경)이다.

4. Node.js는 웹서버와 같이 확장성 있는 네트워크 프로그램을 제작하기 위해 만들어졌다.

자바스크립트를 활용하며 Non-blocking I/O와 단일 스레드 이벤트 루프를 통해 처리하는 특징이 있다.
