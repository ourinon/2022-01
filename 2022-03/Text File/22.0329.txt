- python 2 의 최종번전은 2.7이며 더이상 유지보수 되지않는다.
- 그에반해 python3는 새롭게 나왔으며 계속 버전이 증가하는 추세이다.
- python 3의 기본 저장 방식은 Unicode이다.
. 기존 python 2.x 버전은 ASCII 방식으로 저장되었으며, 유니코드 방식을 지원하기는
하지만 유니코드 방식으로 저장하려면 별도의 설정이 필요했다.

- python3 는 정수 처리 기능이 개선되었다.
ex) 기존 버전은 2.5의 결과를 얻기위해선 실수형태로 5.0/2.0으로 입력해야했으나,
3버전 에서는 생각하는 그대로 원하는 결과가 나온다. 5/2 = 2.5


delegate 란?

- 메서드 대리자이다.

- 메서드 대리자를 선언해준뒤에 원하는 메서드를 참조 시킬수있다.

즉, 함수를 보관하는 통을 만들고(대리자 선언) 그 안에 함수를 넣고 

통을 통째로 가져와서 함수를 실행시킨다.

&&&&&&&&&&&&&&&&&&&&&&&&
ex)
delegate void Del(); // 대리자 생성 (통)

- 매개변수가없으며 반환형이 void형의 메서드만 참소시킬수있다.

delegate int Del(int a) 이라면

- 매개변수가 int형이며 반환형이 int인 메서드를 참조할수있다.

즉, delegate의 반환형과 매개변수 자료형이 동일한 메서드만 참조가능하다.

public void Main() {

	Del myDel; // 대리자 객체 생성

	myDel = print; // Print 라는 메서드를 참조한다.
	
	myDel() // 대리자를 사용해서 메서드 호출
}
